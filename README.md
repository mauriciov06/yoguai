## Yoguai

Plataforma desarrollada en Laravl 5.8, para realizar pedidos online de yogurt y sistema de administración.

## Funcionalidades

- CRUD de Usuarios
- CRUD de Clientes
- CRUD de Sabores
- CRUD de Productos
- CRUD de Proveedores
- Listado de Pedidos
- Filtros de búsqueda para cada uno de los modulos.
- Landing Page 
- Inicio de sesión
- Registro de usuario
